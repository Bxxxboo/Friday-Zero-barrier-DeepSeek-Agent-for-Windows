【系统要求】
  · Windows 10 或 Windows 11（64 位）
  · 能连接互联网（首次运行会自动补齐部分组件；对话需要 DeepSeek API）
  · 无需手动安装 Python（本安装包已内置运行环境）
  · .NET Framework 4.7.2+（Win10/11 通常已自带，程序启动时会检测）

【推荐：零门槛安装（3 步）】
  1. 右键 zip → 属性 → 若有「解除锁定 / Unblock」请勾选 → 确定
  2. 解压到任意文件夹（建议纯英文路径，例如 D:\Friday）
  3. 双击「Friday-Setup-版本号.exe」，按安装向导完成安装
     · 默认安装到：%LOCALAPPDATA%\Programs\Friday
     · 可勾选「安装完成后启动星期五」
     · 首次启动会自动检测并安装 WebView2、VC++ 运行库（需联网，约 1～5 分钟）

【第一步：解压】
  1. 右键压缩包 → 属性 → 若底部有「解除锁定」或「Unblock」，请勾选后确定
     （从 Gitee/GitHub 下载的文件建议做这一步）
  2. 右键压缩包 →「全部解压缩」或「解压到当前文件夹」
  3. 得到文件夹，里面应有：
       · 本文件「安装教程.txt」
       · 「Friday-Setup-版本号.exe」（安装程序，双击运行）
       · （可选）「解除锁定.ps1」——若 exe 被 SmartScreen 拦截可尝试运行
  4. 若浏览器直接下载了 .exe 而非 zip，可跳过解压，直接运行该 exe

【第二步：运行安装程序】
  1. 双击「Friday-Setup-版本号.exe」
  2. 若 Windows 提示「未知发布者」或 SmartScreen 拦截：
       点击「更多信息」→「仍要运行」（本软件未购买代码签名证书）
     或右键「解除锁定.ps1」→ 使用 PowerShell 运行后，再双击安装程序
  3. 按向导选择安装位置（默认即可），完成安装
  4. 从开始菜单或桌面快捷方式启动「星期五」
  5. 首次启动会自动：
       · 解除程序目录锁定（Zone.Identifier）
       · 检测并安装 WebView2 运行时（Win10 常见缺失项）
       · 检测并安装 VC++ 2015–2022 运行库
     若自动安装失败，可手动安装 WebView2：
       https://developer.microsoft.com/microsoft-edge/webview2/
       选「Evergreen Standalone Installer」

【第三步：首次设置（约 3 分钟）】
  启动后会自动出现引导：
  1. 填写 DeepSeek API Key
     · 打开 https://platform.deepseek.com 注册并创建 API Key
     · 复制 Key 粘贴到星期五设置中，点保存/测试
  2. 选择「默认操作文件夹」
     · 建议选「文档\星期五」（程序可自动创建）
  3. （可选）启用视觉辅助 — 用于识图、分析截图
     · 设置 → API 连接 → 视觉辅助模型
     · 勾选「启用视觉辅助」
     · 填写豆包/火山引擎 Ark API Key
     · 「视觉模型/端点」必须填 ep- 开头的推理接入点 ID（不是裸模型名）
     · 点「测试视觉 API」，成功后会自动保存
     · 底部状态栏应显示「视觉 在线」
  4. （可选）启用生图 — 设置 → API 连接 → 生图
     · 配置 OpenAI 兼容或火山方舟 API Key 与模型名
     · 生成的图片会保存在会话历史中，重新打开对话仍可查看
  5. （可选）微信端 AI — 设置 → 微信端 AI → 一键配置
     · 自动安装 Node.js、OpenClaw、微信插件（需联网，约 3～5 分钟）
     · 完成后自动弹出微信扫码窗口
  6. 完成引导后即可开始对话

【第四步：创建桌面快捷方式（可选）】
  安装向导可勾选创建桌面快捷方式；也可从开始菜单固定到任务栏。

【日常使用】
  · 从开始菜单或桌面快捷方式打开「星期五」
  · 在对话框输入需求，例如：
      「帮我看看 C 盘还剩多少空间」
      「整理一下下载文件夹」
  · 设置、历史对话保存在本机，路径：
      C:\Users\你的用户名\AppData\Roaming\Friday\

【从旧电脑迁移数据（可选）】
  若之前在另一台电脑用过星期五，想带走对话和设置：
  1. 在旧电脑复制整个文件夹：
       C:\Users\旧用户名\AppData\Roaming\Friday\
  2. 粘贴到新电脑相同位置（覆盖前请先备份）
  3. 必须同时包含 settings.json 和 .fernet_key 两个文件
  4. 打开星期五 → 设置 → 检查「默认操作文件夹」是否为本机有效路径

【高级功能：Python 数据分析（大多数用户可跳过）】
  若需要使用「运行 Python 脚本 / pandas 数据分析」：
  1. 打开星期五 → 设置 → 初始化 Python 环境
  2. 程序会自动尝试 winget 安装 Python，或下载便携 Python 到 AppData（需联网）
  3. 等待安装完成即可
  （也可手动安装 Python 3.12：https://www.python.org/downloads/）

【常见问题】
  Q：杀毒软件报毒？
  A：PyInstaller 打包的程序可能被误报，可添加信任/白名单。

  Q：测试 API 时提示「网络错误」或无法连接？
  A：① 这通常指连不上本机后台服务，不是 DeepSeek 网站本身
     ② 完全关闭应用后重开；检查杀毒/防火墙是否拦截 星期五.exe
     ③ 查看 %APPDATA%\Friday\friday.log 是否有报错
     ④ 若提示 DeepSeek 相关错误，再检查 API Key、账户余额与能否访问 api.deepseek.com

  Q：界面一直显示「正在连接…」、消息排队不执行？
  A：① 完全退出 星期五.exe（任务管理器确认无残留）后重开
     ② 关闭 VPN/代理，或把 127.0.0.1 加入代理绕过列表
     ③ Windows 防火墙首次提示时选「允许访问」
     ④ 杀毒软件添加 星期五.exe 信任/白名单
     ⑤ 查看 %APPDATA%\Friday\friday.log 末尾是否有报错
     ⑥ 确认已填写并保存 DeepSeek API Key（设置 → API 连接）

  Q：启动时报 Python.Runtime / pythonnet 错误？
  A：① 运行「解除锁定.ps1」或重新运行安装程序修复安装
     ② 确认安装目录为纯英文路径（默认 %LOCALAPPDATA%\Programs\Friday 即可）
     ③ 在「启用或关闭 Windows 功能」中勾选 .NET Framework 4.8
     ④ 让程序自动安装 WebView2，或手动安装后重试

  Q：Win10 白屏 / 窗口打不开？
  A：首次启动会自动安装 WebView2；若失败请手动安装 Evergreen 版后重试。

  Q：提示端口被占用？
  A：关闭其他正在运行的星期五窗口，或重启电脑后再开。

  Q：API Key 无效？
  A：检查 Key 是否完整、账户是否有余额、网络是否正常。

  Q：视觉 API 测试通过但无法识图？
  A：① 确认已勾选「启用视觉辅助」且测试后已自动保存
     ② 端点必须是 ep- 开头的视觉理解接入点
     ③ 粘贴截图后发送，或说「帮我看这张截图」
     ④ 使用 Agent 模式（Ask 模式仅只读，识图可用但别指望执行写操作）

  Q：换电脑后对话没了？
  A：对话不在安装文件夹里，需按上文「迁移数据」拷贝 AppData\Friday。

  Q：升级后有什么新功能？
  A：设置 → 安全与更新 → 查看更新历史；或启动时会自动弹出未读版本说明。

【技术支持】

  出问题时可将该文件发给开发者排查。

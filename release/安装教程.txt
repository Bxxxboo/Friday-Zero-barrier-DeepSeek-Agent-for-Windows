【系统要求】
  · Windows 10 或 Windows 11（64 位）
  · 能连接互联网（对话需要 DeepSeek API）
  · 无需安装 Python（本安装包已内置运行环境）
  · 已启用 .NET Framework 4.7.2 或更高（Win10/11 通常已自带）
  · 已安装 WebView2 运行时（Win11 通常已内置）

【第一步：解压】
  1. 右键压缩包 →「全部解压缩」或「解压到当前文件夹」
  2. 得到文件夹，里面应有：
       · 本文件「安装教程.txt」
       · 文件夹「Friday」（内含 星期五.exe）
  3. 建议把整个「Friday」文件夹放到**纯英文路径**，例如：
       D:\Friday
     或 D:\Apps\Friday
     （不要只拷贝单个 exe，必须保留整个「Friday」文件夹及其 _internal 目录）

【第二步：首次运行】
  1. 进入「Friday」文件夹，双击 星期五.exe
  2. 若 Windows 提示「未知发布者」：
       点击「更多信息」→「仍要运行」（本软件未购买代码签名证书）
  3. 若窗口打不开、一直白屏（多见于 Win10）：
       请先安装 WebView2 运行时（微软官方，免费）：
       https://developer.microsoft.com/microsoft-edge/webview2/
       选「 Evergreen Standalone Installer」下载安装后重试

【第三步：首次设置（约 3 分钟）】
  启动后会自动出现引导：
  1. 填写 DeepSeek API Key
     · 打开 https://platform.deepseek.com 注册并创建 API Key
     · 复制 Key 粘贴到星期五设置中，点保存/测试
  2. 选择「默认操作文件夹」
     · 建议选「文档\星期五」（程序可自动创建）
  3. （可选）启用视觉辅助 — 用于识图、分析截图
     · 设置 → API 连接 → 视觉辅助模型
     · 勾选「启用视觉辅助」
     · 填写豆包/火山引擎 Ark API Key
     · 「视觉模型/端点」必须填 ep- 开头的推理接入点 ID（不是裸模型名）
     · 点「测试视觉 API」，成功后会自动保存
     · 底部状态栏应显示「视觉 在线」
  4. （可选）启用生图 — 设置 → API 连接 → 生图
     · 配置 OpenAI 兼容或火山方舟 API Key 与模型名
     · 生成的图片会保存在会话历史中，重新打开对话仍可查看
  5. 完成引导后即可开始对话

【第四步：创建桌面快捷方式（可选）】
  方式 A — 手动：
    右键 星期五.exe → 发送到 → 桌面快捷方式

  方式 B — 脚本（与压缩包同目录）：
    右键「创建桌面快捷方式.ps1」→ 使用 PowerShell 运行
    若提示无法运行，先在 PowerShell 中执行：
    Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

【日常使用】
  · 双击 星期五.exe 打开
  · 在对话框输入需求，例如：
      「帮我看看 C 盘还剩多少空间」
      「整理一下下载文件夹」
  · 设置、历史对话保存在本机，路径：
      C:\Users\你的用户名\AppData\Roaming\Friday\

【从旧电脑迁移数据（可选）】
  若之前在另一台电脑用过星期五，想带走对话和设置：
  1. 在旧电脑复制整个文件夹：
       C:\Users\旧用户名\AppData\Roaming\Friday\
  2. 粘贴到新电脑相同位置（覆盖前请先备份）
  3. 必须同时包含 settings.json 和 .fernet_key 两个文件
  4. 打开星期五 → 设置 → 检查「默认操作文件夹」是否为本机有效路径

【高级功能：Python 数据分析（大多数用户可跳过）】
  若需要使用「运行 Python 脚本 / pandas 数据分析」：
  1. 安装 Python 3.12：https://www.python.org/downloads/
     安装时勾选「Add python.exe to PATH」
  2. 打开星期五 → 设置 → 初始化 Python 环境
  3. 等待安装完成即可

【常见问题】
  Q：杀毒软件报毒？
  A：PyInstaller 打包的程序可能被误报，可添加信任/白名单。

  Q：测试 API 时提示「网络错误」或无法连接？
  A：① 这通常指连不上本机后台服务，不是 DeepSeek 网站本身
     ② 完全关闭应用后重开；检查杀毒/防火墙是否拦截 星期五.exe
     ③ 查看 %APPDATA%\Friday\friday.log 是否有报错
     ④ 若提示 DeepSeek 相关错误，再检查 API Key、账户余额与能否访问 api.deepseek.com

  Q：启动时报 Python.Runtime / pythonnet 错误？
  A：① 确认安装路径为纯英文（如 D:\Friday\Friday\）
     ② 在「启用或关闭 Windows 功能」中勾选 .NET Framework 3.5（含 4.x）
     ③ 安装 WebView2 运行时后重试

  Q：提示端口被占用？
  A：关闭其他正在运行的星期五窗口，或重启电脑后再开。

  Q：API Key 无效？
  A：检查 Key 是否完整、账户是否有余额、网络是否正常。

  Q：视觉 API 测试通过但无法识图？
  A：① 确认已勾选「启用视觉辅助」且测试后已自动保存
     ② 端点必须是 ep- 开头的视觉理解接入点
     ③ 粘贴截图后发送，或说「帮我看这张截图」
     ④ 使用 Agent 模式（Ask 模式仅只读，识图可用但别指望执行写操作）

  Q：换电脑后对话没了？
  A：对话不在安装文件夹里，需按上文「迁移数据」拷贝 AppData\Friday。

  Q：升级后有什么新功能？
  A：设置 → 安全与更新 → 查看更新历史；或启动时会自动弹出未读版本说明。

【技术支持】
  日志文件：%APPDATA%\Friday\friday.log
  出问题时可将该文件发给开发者排查。

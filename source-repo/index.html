星期五 源仓库

Windows AI 电脑管家 · 用于 GitHub「Import repository」的 Git 源地址

① 复制源仓库地址

将下方地址粘贴到 GitHub 导入页的 The URL for your source repository 输入框。

打开 GitHub 导入页 浏览源仓库网页

② 如何生成源地址（推荐 Gitee 中转）

  1. 安装 Git，注册 Gitee 并创建私人令牌
  2. 在项目根目录运行：
    $env:GITEE_TOKEN='令牌'; scripts\push-gitee-source.cmd -GiteeUser 你的用户名
  3. 脚本会推送代码到 Gitee，并更新本页的克隆地址
  4. 导入前确认 Gitee 仓库为公开（私有仓库 GitHub 无法拉取会报 Authentication failed）
  5. 打开上方「GitHub 导入页」，粘贴地址（不要末尾斜杠），Owner 填 Bxxxboo，Repository name 填 Friday-Zero-barrier-DeepSeek-Agent-for-Windows
  6. 导入完成后运行 scripts/publish-release.ps1 同步双端 Release

③ 本地预览本页

双击本文件，或运行 powershell -File scripts/serve-source-repo.ps1 在浏览器打开。

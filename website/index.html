星期五 你的 Windows AI 电脑管家

用自然语言交代任务，星期五会在本机规划步骤、调用工具完成。对话、设置、电脑体检，都在你的 Windows 上。

星期五能做什么

不是只会聊天的网页窗口。你描述目标，Agent 规划步骤、调用本机工具执行，结果回到对话里。

听懂并规划

自然语言描述任务，Agent 拆解步骤、选择工具，复杂流程不用自己写脚本。

本机真正执行

文件整理、系统体检、文档生成、截屏识图等 28+ 工具在 Windows 上运行，不依赖云端代操作。

你始终掌控

Ask / Agent / Yolo 多种模式；危险操作需确认；设置与会话保存在本机 %APPDATA%\Friday\。

核心场景：微信远程部署任务

不在家也能用手机微信下发任务：整理、生图、脚本一键部署到家中电脑。执行在本机完成，进度与结果回到聊天；删除或改写前在聊天里审批。

  • 手机微信下发，电脑自动执行
  • 进度与结果回到微信，不在电脑前也能跟进
  • 危险操作在聊天里审批后再动手

扩展能力

概览之外，还可以按你的习惯继续配置。

多模型可配置

DeepSeek、火山方舟、小米 MiMo 等 API 自选，按场景切换对话与推理模型。

技能与快捷指令

输入 / 选技能，Ctrl+V 粘贴截图；常用任务不用每次重讲一遍。
